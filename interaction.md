# 个人网站交互设计方案

## 核心交互功能

### 1. 技能可视化展示
- **技术栈引力网络**：创建一个动态的3D网络图，展示您的技术栈
  - 中心节点：LLM架构优化、强化学习、CUDA计算优化
  - 分支节点：具体技术点如Transformer、PPO、FlashAttention等
  - 交互：鼠标悬停显示技术详情，点击展开相关项目

### 2. 项目时间线
- **研究历程时间轴**：交互式时间线展示您的项目演进
  - 从GlowFlow到YingHub系列项目的发展轨迹
  - 每个节点可点击查看项目详情、技术栈、GitHub链接
  - 支持筛选：按技术类型（LLM/RL/CUDA）、项目状态分类

### 3. 学术成就展示
- **论文影响力可视化**：
  - AAAI 2026论文的评审过程可视化
  - Committee成员身份展示
  - 参与竞赛成绩展示（如ARC Prize 2025）

### 4. 技术博客聚合
- **CSDN文章动态展示**：
  - 最新博客文章轮播展示
  - 按技术标签分类显示
  - 阅读量和互动数据可视化

## 页面导航交互
- 平滑页面切换动画
- 导航栏高亮当前页面
- 移动端响应式汉堡菜单

## 特殊效果
- 代码雨背景效果（代表编程能力）
- 神经网络连接动画
- 技能熟练度动态图表
- 项目卡片悬浮效果
